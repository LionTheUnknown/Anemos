# Anemos

Weather app with functionaliy for city search, current conditions, and 5-day forecast.

## Stack

- **Frontend**: React, Vite, MUI, Tailwind
- **Backend**: Node.js (Bun), Express, PostgreSQL
- **Weather data**: Open-Meteo API

## Instructions

1. Start PostgreSQL: `docker compose up -d`
2. Backend: `cd backend && bun install && bun run dev`
3. Frontend: `cd frontend && bun install && bun run dev`
4. Open http://localhost:5173
